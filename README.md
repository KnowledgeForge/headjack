## Install Using Pip

```sh
pip install headjack
```

## Start the Headjack Server

```
headjack
```

# Lint, Test, and Check Coverage

This project includes a Makefile for use with [GNU Make](https://www.gnu.org/software/make/).

Run linters.
```
make lint
```

Run tests.
```
make test
```

Run tests with coverage.
```
make coverage
```
